## Runtime policy

- CC0 Quaternius modular skinned assets are enabled by default in the manifest helper.
- CC-BY Sketchfab assets are copied and listed but disabled by default, so credits are preserved before you choose to use them.
- Static/non-skinned full-scene clothing assets should be split, scaled, and rigged before production use on animated Volex bodies.
